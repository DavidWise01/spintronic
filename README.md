# spintronic

SPINTRONIC — a bit stored as a direction of spin: a spin-transfer-torque MRAM cell, the magnetization switched by a spin-polarized current (Landau-Lifshitz-Gilbert + Slonczewski STT). The fourth information carrier {charge, light, lattice, spin}, completing the tensor. Live 2D-canvas, no external deps. HEUREMA. ROOT0, with AVAN. CC-BY-ND-4.0.
